# Music Platform
REST API для работы с музыкой.

## Для запуска необходимо

1. git clone https://github.com/aaalis/MusicPlatform
2. cd MusicPlatform
3. docker-compose up

## Тестирование

:heavy_check_mark: localhost:8000/api/doc

## Стек технологий

- PHP 8.2
- Laravel 10
- PostgreSQL 14
- Docker
